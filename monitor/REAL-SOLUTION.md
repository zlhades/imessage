## 📊 完整工作流程

### 阶段 1: 信息不足（WAITING_INFO）

```
📬 发现新消息
📊 分析：信息不足
📝 生成追问草稿
   → 用户确认
   → 复制到 AI 对话
```

### 阶段 2: 需要调查（INVESTIGATING）

```
📬 发现新消息（用户补充了信息）
📊 分析：需要调查
📝 生成调查建议
   → 用户确认
   → 复制到 AI 对话
```

### 阶段 3: 准备回复（READY_TO_REPLY）

```
📬 发现新消息（信息完整）
📊 分析：信息完整
📝 生成回复草稿
   → 用户确认/修改
   → 复制到 AI 对话
```

### 阶段 4: 事件结束（READY_TO_CLOSE）

```
📬 发现新消息（包含"已修复"）
📊 分析：检测到结束标记
📝 生成事件总结
   → 用户确认
   → 完成
```
