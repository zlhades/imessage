## ⚠️ 注意事项

### 优点
- ✅ Qwen 会持续监听
- ✅ 可以在对话中实时看到执行结果
- ✅ 可以随时打断 Qwen 提问

### 缺点
- ⚠️ Qwen 会一直占用对话，无法同时做其他事
- ⚠️ 如果关闭 Qwen Code，监听会停止
- ⚠️ 每次轮询都会消耗 Token（调用 MCP 工具）

### Token 消耗

每次轮询：
- 读取消息：~100 tokens
- 分析执行：~500-2000 tokens（如果有指令）
- 无指令时：~200 tokens

**每小时约消耗**: 1000-5000 tokens（取决于消息频率）

## 🛑 停止监听

在 Qwen Code 中说：

```
停止监听
```

或按 `Ctrl+C` 中断。
