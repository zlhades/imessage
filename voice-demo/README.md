# 🎤 AI 语音输入 PWA

> 持续监听 → 实时转文字 → 自动清理语气词
> 
> **PWA 版本**：支持安装到桌面/手机，离线可用

---

## 快速开始

### 1. 设置 API Key（可选）

```bash
export DASHSCOPE_API_KEY="your_api_key_here"
```

> 获取 API Key: https://dashscope.console.aliyun.com/
> 新用户免费额度：1M tokens
> **不设置也可使用**，会自动切换到本地清理模式

### 2. 运行

```bash
cd voice-demo
python3 app-light.py
```

### 3. 打开浏览器

| 设备 | 访问方式 |
|------|----------|
| 电脑 | **http://localhost:8765** |
| 手机 | **http://[你的电脑IP]:8765** |

> ⚠️ 语音识别需要 **Chrome/Edge/Safari 16.4+**

---

## PWA 安装

### 电脑（Chrome/Edge）
1. 打开 `http://localhost:8765`
2. 点击地址栏右侧的 **安装** 图标
3. 或：点击右上角菜单 → **安装语音输入**

### iPhone（Safari）
1. 打开 `http://[你的电脑IP]:8765`
2. 点击底部 **分享** 按钮
3. 选择 **"添加到主屏幕"**

### Android（Chrome）
1. 打开 `http://[你的电脑IP]:8765`
2. 点击右上角 **⋮** 菜单
3. 选择 **"安装应用"** 或 **"添加到主屏幕"**

## 功能说明

| 功能 | 说明 |
|------|------|
| 🎙️ **持续监听** | 打开页面自动录音，直到你停止 |
| 🔄 **实时转写** | 说话同时显示文字，无需等待 |
| ✨ **AI 清理** | Qwen 去除语气词、添加标点（需 API Key） |
| 📝 **本地清理** | 无 API Key 时用正则规则清理（免费） |
| 📋 **一键复制** | 复制清理后的文本 |
| 📲 **PWA 安装** | 安装到桌面/手机，随时使用 |

### 重要限制
- ⚠️ **必须用户点击才能启动麦克风**（浏览器安全策略）
- ⚠️ **切后台/锁屏会断开录音**
- ⚠️ **需要网络连接**（语音识别在云端处理）

## 技术栈

| 模块 | 技术 |
|------|------|
| 后端 | Python 3（标准库） |
| 前端 | 纯 HTML + CSS + JS |
| PWA | Service Worker + Manifest |
| 语音识别 | 浏览器 Web Speech API |
| AI 清理 | 通义千问 qwen-turbo |
| 本地清理 | 正则表达式规则 |

**零第三方依赖！**
